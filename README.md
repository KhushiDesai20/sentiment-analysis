# Sentiment Analysis Project

This project analyzes YouTube comments using NLP techniques to understand viewer sentiment, detect toxicity, and generate meaningful summaries for better content insights.

## Features
- Fetches YouTube video comments using YouTube Data API

- Performs sentiment analysis (Positive / Negative / Neutral)

- Detects toxic or harmful comments

- Summarizes large volumes of comments using batch-wise   summarization

- Generates an overall insight summary of viewer engagement

- Visualizes results using charts (sentiment distribution)

## Technologies Used

- Python

- YouTube Data API

- Hugging Face Transformers

- FLAN-T5 – Sentiment & toxicity classification

- BART-Large-CNN – Comment summarization

- LangChain – Prompt handling & LLM chaining

- Matplotlib & NLTK – Visualization & text processing

## Output
- Pie chart / graphs showing sentiment distribution (Positive, Negative & Neutral)
- Bar graph showing Toxic VS Non-Toxic classification
- Pie chart showing how much negative keywords & other keywords extracted.
